Support

We are here to give you the support you need

We offer three support packages, so there is always a perfect fit for your situation.

Support

Our open source products are developed and maintained for multiple customers, resulting in new features developed for and issues resolved for these customers. Our support agreements offers a possibility to take advantage of these changes and improvements in the application.

Eveoh also provides maintenance to support future versions of back-end products, like Scientia Syllabus Plus, Microsoft SharePoint or Blackboard, and future browser versions. This way, the application is future proof and no problems will arise when upgrading.

Recommended support package

We recommend the software assurance package for most of the customers. With this package, an 8x5 helpdesk is available, incidents will be resolved free of charge, and the customer will profit from future developments of the application.

Available support packages

Basic support

The basic support package includes application management, where the following services will be delivered for a fixed price:

  • Receiving and administering calls
  • Third-line support
  • Analysis of incidents
  • Resolving incidents (under terms of warranty)
  • Availability of a development environment

If desired, Eveoh will also carry out system administration of the production environment for Oracle Java and Apache Tomcat.

The following application management services will be billed using post calculation:

  • Compiling and testing new releases of the application
  • Development of new features
  • Configuration changes at the explicit request of the customer

The customer needs to take care of first-line support and second-line support (eliciting of additional functional requirements).

Software assurance recommended

MyTimetable is developed for multiple customers, resulting in the addition of new features and resolution of incidents. With the software assurance package, you can benefit from these changes. The software assurance package consists of:

  • All services from the basic service package
  • Compiling, configuring, testing and deploying new releases of the software
  • Resolving incidents is included in this service package free of charge

The following services will be billed using post calculation:

  • Development of new features
  • Configuration changes at the explicit request of the customer

The customer needs to take care of first-line support and second-line support (eliciting of additional functional requirements).

All-in

The all-in package covers all services of the basic and software assurance service packages, complemented with free of charge configuration changes and a 15% discount on the hourly rate for the implementation of new features.

Configuration changes are, for example, changes in the corporate styling of MyTimetable, enabling or disabling functionality, or support for other methods of timetabling.
