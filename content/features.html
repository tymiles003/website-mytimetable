--- title: Features ---

Features

Offer personalised timetables

MyTimetable allows users to fully personalise their timetable. Depending on the exact wishes of the institution, students and staff can add and remove timetables, for example for courses, study programmes or staff members. They can also select the student sets they are in, even further personalising their timetable.

Timetables can be added automatically using the integrations we offer. It is, for example, possible to use course enrolments from your student information system as an additional source of information.

Plaatje

Views for every use case

Students and staff will be familiar with and appreciate the many graphical views MyTimetable offers. Timetables can be shown per day, week, or month, or using a list view for date ranges such as semesters.

Colours are used to visualise different activity types. Overlap between activities can be highlighted, so students can use MyTimetable to compile a clash-free timetable before enrolling for courses.

Enable your support staff

Support staff can assist students and teachers by quickly looking up timetables. Room and building timetables can be viewed in several formats.

An availability view allows for a birds-eye view of several rooms. It can be used to locate unoccupied rooms, or to swiftly resolve practical problems.

Plaatje

Synchronise with third-party calendars

Personalised timetables can be published to Microsoft Office 365, Microsoft Exchange or Google Calendar using push synchronisation, providing up-to-date calendars for staff and students. Staff timetables become part of their calendar, resulting in complete and correct availability information when making appointments with co-workers.

A pull-based synchronisation using iCalendar feeds is available for calendar applications such as Microsoft Outlook, Google Calendar, Apple Calendar, and various smartphone calendar applications.

Mobile supported out-of-the-box

A clean and lightweight mobile interface is automatically shown when using a mobile phone. MyTimetable is available in your mobile browser, so there's no need to install an app.

Downloads in many formats

Timetables can be downloaded in multiple formats, such as printable PDFs, making it easy to print your timetable while retaining the great layout of MyTimetable.

Other formats, such as CSV and iCalendar files are also supported.

Open API

A fully RESTful API is available to reuse timetabling data and user profile data in portals, apps, and other applications. Data is returned in JSON or XML formats. The API is secured using the OAuth 2 protocol, putting the end-users in control of their data.

Security

Timetabling data can be fully secured using a role-based access model. For example, staff or draft timetables can be hidden from students when necessary, or students can be locked down to their personal timetable.

Feature overview

Timetables

  • Personal schedule per user
  • Location schedules with availability view
  • Activity types, which can be used to colour or filter events
  • Draft activities, which can be hidden for certain roles
  • Custom date ranges, such as semesters
  • Custom week names/numbers to match your academic calendar
  • Multiple data sources (per year, per department, or both)

Push synchronisation

  • Microsoft Exchange
  • Microsoft Office 365
  • Google Apps for Education
  • Google Calendar

Export formats

  • PDF (week, list)
  • Comma separated values (CSV)
  • iCalendar

Interface

  • Mobile interface for phones and tablets
  • Custom branding
  • Localisation

Security

  • Role-based access to timetables
  • Integration with any authentication system

Management

  • Provisioning API and toolkit available
  • Web-based administration panel
